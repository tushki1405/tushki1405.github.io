About

Daily Expense Manager is a desktop based application which helps you to manage your daily expenses. When studying in college, living with roommates, going for trips, etc there are always expenses that are individual, and there are always some expenses that have to be shared among all. Though individual expenses are relatively easy to manage, calculating shared expenses is a bit tedious task. This application will help you manage all expenses very easily. In addition there are a lot of other features which will make all money management tasks a breeze.

You must be wondering "Why a desktop application and not a web version?". This is because of my personal experiences where though I was an engineering student and now a professional in IT industry, I had to and still have to go days without internet access. But I used to have my laptop and that is all I needed to update all the details everyday.

All the data is saved in MSAccess files. Multiple users can use the same application by creating their own individual files. Both application and data files are portable and can be easily transferred.

 
Features

SIMPLIFIED REPORTS
Get all the frequently required information at home window. Access reports easily for:
  • Total Expenses for the last week and month displayed as graph.
  • Current balance in accounts.
  • All data related to last 10/20 transactions.
  • Payments and receivables information.
MANAGE ACCOUNTS
Manage your accounts easily here. Below features available for management:
  • Create New accounts from which transactions are made.
  • Add/Reduce amount from Accounts.
  • Transfer amount from one account to another.
  • Delete Account.
  • Rename Account.
EDIT EXPENSES
Individual Expenses can be edited easily.
  • Easily find the expenses for a particular day.
  • View all the information for the expenses.
  • Edit the individual expense.
  • Directly update the amount for the selected account.
VIEW EXPENSES
View all the expenses between a date range.
  • Select a date range to view expenses between that date.
  • Graph displayed for category-wise total expenses between date range.
  • Individual expenses displayed for the same range.
  • Sort expenses according to date, amount, category, etc.
SHARED EXPENSES
Easily Manage Expenses that are shared in a group.
  • Create a new Title for new group Expenses.
  • Add participants any time.
  • Easily add expenses and enter share of each person.
  • Delete individual as well as whole shared expense title.
SHARED EXPENSES
View the final calculations in the form of reports for shared expenses.
  • Data is saved in MSAccess file which can be sent to friends.
  • Persons overall payment/receivable displayed in graph.
  • Recommended payment calculation displayed for easy exchange.
